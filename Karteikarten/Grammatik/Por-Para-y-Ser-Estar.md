# por para y ser estar

Wann benutzt man `por` statt `para` als Grundregel? :: wenn der Satz zurückschaut: Ursache, Grund, Motiv, Mittel oder Weg<br>Kontrast: `para` schaut nach vorne auf Ziel, Zweck oder Empfänger. [[Por-y-Para]]

Welche Frage passt für Zweck? `¿___ estudias español?` :: Para qué<br>`para qué` fragt nach Ziel/Zweck; `por qué` fragt nach Grund/Ursache. [[Por-y-Para]]
<!--ID: 1777000000017-->

Welche Form passt? `Lo explico para que lo ___.` :: entiendas<br>`para que` + anderes Subjekt -> subjuntivo. [[Por-y-Para]]

Grundkontrast `ser` vs. `estar` :: `ser` = was etwas grundsätzlich ist; `estar` = wie/wo etwas gerade ist<br>[[Ser-y-Estar]]
<!--ID: 1777000000019-->

Welche Form passt? `La reunión ___ en la biblioteca.` / Gegenbeispiel: `Estoy en la biblioteca.` :: es<br>Ort eines Ereignisses -> `ser`; physischer Aufenthaltsort -> `estar`. [[Ser-y-Estar]]

Was ist der Unterschied? `ser rico` / `estar rico` :: reich sein / lecker sein<br>[[Ser-y-Estar]]
