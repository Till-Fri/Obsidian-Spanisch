# Obsidian_to_Anki
# basiszeiten

presente: typische Verwendung :: aktuelle Zustände, Gewohnheiten und allgemeine Wahrheiten
pretérito perfecto: typische Verwendung :: abgeschlossene Handlung mit Bezug zur Gegenwart oder zu einem noch offenen Zeitraum
pretérito indefinido: typische Verwendung :: abgeschlossene, punktuelle Handlung in einem klar abgeschlossenen Zeitraum

yo (tener, presente) :: tengo
tú (tener, presente) :: tienes
él/ella/usted (tener, presente) :: tiene
nosotros (tener, presente) :: tenemos
vosotros (tener, presente) :: tenéis
ellos/ellas/ustedes (tener, presente) :: tienen

yo (haber + participio, pretérito perfecto) :: he + participio
él/ella/usted (haber + participio, pretérito perfecto) :: ha + participio
nosotros (haber + participio, pretérito perfecto) :: hemos + participio
ellos/ellas/ustedes (haber + participio, pretérito perfecto) :: han + participio

1. Person Singular von asumir im indefinido :: asumí
3. Person Singular von asumir im indefinido :: asumió
3. Person Plural von asumir im indefinido :: asumieron

Welche Zeit passt? "Esta semana el club ya ___ los primeros pasos." :: pretérito perfecto<br>Ej.: esta semana el club ya ha dado los primeros pasos. [[Rebajas para Lewandowski]]
Welche Zeit passt? "La semana pasada el ministro ___ el cargo." :: pretérito indefinido<br>Ej.: la semana pasada el ministro asumió el cargo. [[Noticias]]
Welche Zeit passt? "El gobierno ___ un nuevo plan." :: presente<br>Ej.: el gobierno lanza un nuevo plan. [[Noticias]]
Welche Zeit passt? "La decisión ___ en sus manos." :: presente<br>Ej.: la decisión está en sus manos. [[Rebajas para Lewandowski]]
