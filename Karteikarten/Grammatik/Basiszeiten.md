# Obsidian_to_Anki
# basiszeiten

Welche Zeit passt? `El gobierno ___ un nuevo plan.` / Gegenbeispiel: `La semana pasada el gobierno ___ un nuevo plan.` :: lanza / lanzó<br>Aktuelle Aussage oder Gewohnheit -> presente. Abgeschlossene Vergangenheit -> indefinido. [[Noticias]]

Welche Zeit passt? `La decisión ___ en sus manos.` / Gegenbeispiel: `La decisión ___ en sus manos ayer por la noche.` :: está / estuvo<br>Gegenwärtiger Zustand -> presente. Abgeschlossener vergangener Zustand -> indefinido. [[Rebajas para Lewandowski]]

Welche Zeit passt? `Esta semana el club ya ___ los primeros pasos.` / Gegenbeispiel: `La semana pasada el club ya ___ los primeros pasos.` :: ha dado / dio<br>Offener Zeitraum -> pretérito perfecto. Abgeschlossener Zeitraum -> indefinido. [[Rebajas para Lewandowski]]

Welche Zeit passt? `En los últimos meses el rendimiento ___ dudas.` / Gegenbeispiel: `En 2024 el rendimiento ___ dudas.` :: ha generado / generó<br>Vergangenheit mit Gegenwartsbezug -> pretérito perfecto. Abgeschlossener Vergangenheitsrahmen -> indefinido. [[Rebajas para Lewandowski]]

Welche Zeit passt? `El ministro ___ el cargo la semana pasada.` / Gegenbeispiel: `El ministro ya ___ el cargo esta semana.` :: asumió / ha asumido<br>Abgeschlossene Zeitangabe -> indefinido. Noch offener Zeitraum -> pretérito perfecto. [[Noticias]]

Welche Zeit passt? `El proyecto ___ el éxito pero fracasó al final.` / Gegenbeispiel: `El proyecto casi siempre ___ el éxito.` :: rozó / roza<br>Einmalige abgeschlossene Handlung -> indefinido. Gewohnheit oder generelle Aussage -> presente. [[Noticias]]
