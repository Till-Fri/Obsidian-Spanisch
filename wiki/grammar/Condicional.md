# condicional

## uso principal
El condicional se usa para hablar de deseos, hipotesis, consejos, propuestas y peticiones corteses.

Fuente principal: [[Condicional]] y [[Grammatik]].

## usos frecuentes
- deseo: `Me gustaria tener un huerto.`
- hipotesis: `Yo viviria en Espana si pudiera.`
- consejo o recomendacion: `Deberias descansar mas.`
- propuesta: `Podriamos quedar manana.`
- peticion cortesia: `Me podria ayudar?`
- opinion sobre acciones o comportamiento: `Seria mejor esperar.`

## forma regular
Infinitivo + terminaciones:
- yo: `-ia`
- tu: `-ias`
- el/ella/usted: `-ia`
- nosotros: `-iamos`
- vosotros: `-iais`
- ellos/ellas/ustedes: `-ian`

Ejemplos:
- `hablar` -> `hablaria`
- `comer` -> `comeria`
- `vivir` -> `viviria`

## irregulares frecuentes
La terminacion es regular, pero cambia la raiz:
- `caber` -> `cabria`
- `decir` -> `diria`
- `haber` -> `habria`
- `hacer` -> `haria`
- `poder` -> `podria`
- `poner` -> `pondria`
- `querer` -> `querria`
- `saber` -> `sabria`
- `salir` -> `saldria`
- `tener` -> `tendria`
- `valer` -> `valdria`
- `venir` -> `vendria`

## contraste rapido
- [[Futuro Imperfecto]]: prediccion o plan futuro.
- [[Subjuntivo]]: deseo, valoracion o posibilidad en subordinadas.
