# Obsidian_to_Anki
# presente de indicativo (recap)

## regel
presente :: Die einfache Gegenwart, verwendet für aktuelle Zustände, Gewohnheiten und allgemeine Wahrheiten.

## anwendungsbeispiele (recaps)
"El FC Barcelona tiene sobre la mesa una decisión urgente." :: Der FC Barcelona hat eine dringende Entscheidung auf dem Tisch. [[Rebajas para Lewandowski]]
"El polaco sigue teniendo cartel internacional." :: Der Pole hat weiterhin internationales Standing. [[Rebajas para Lewandowski]]
"La decisión está en sus manos." :: Die Entscheidung liegt in seinen Händen. [[Rebajas para Lewandowski]]
