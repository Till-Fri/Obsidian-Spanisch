# nominalizacion

## uso principal
La nominalizacion convierte verbos o adjetivos en sustantivos. Es util para escribir textos mas abstractos y academicos en nivel B2.

Fuentes principales: [[Nominalización]], [[Grammatik]] y [[Spanisch]].

## de verbo a sustantivo
Sufijos frecuentes:
- `-cion`: `explotar` -> `la explotacion`, `proteger` -> `la proteccion`
- `-sion`: `revisar` -> `la revision`
- `-miento`: `levantar` -> `el levantamiento`, `descubrir` -> `el descubrimiento`
- `-encia / -ancia`: `existir` -> `la existencia`, `alternar` -> `la alternancia`
- `-o`: `aumentar` -> `el aumento`, `desarrollar` -> `el desarrollo`
- `-anza`: `esperar` -> `la esperanza`
- `-a`: `luchar` -> `la lucha`, `firmar` -> `la firma`
- `-ura`: `romper` -> `la ruptura`, `leer` -> `la lectura`

## de adjetivo a sustantivo
Sufijos frecuentes:
- `-ez / -eza`: `escaso` -> `la escasez`, `rico` -> `la riqueza`
- `-ura`: `loco` -> `la locura`
- `-idad`: `popular` -> `la popularidad`, `objetivo` -> `la objetividad`

## para que sirve
- resumir procesos: `el aumento de los precios`
- escribir con registro formal: `la proteccion del medioambiente`
- evitar frases demasiado largas con muchos verbos
